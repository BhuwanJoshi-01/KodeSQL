How do I connect to a PostgreSQL database using Python?

2 days ago
3 replies

You can use the psycopg2 library to connect to PostgreSQL. Here's an example:

import psycopg2
conn = psycopg2.connect(
    dbname="your_db",
    user="your_user",
    password="your_password",
    host="localhost"
)
