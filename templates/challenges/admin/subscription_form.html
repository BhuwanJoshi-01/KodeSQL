help Help & Tips

Creating User Subscriptions
  • This assigns a subscription plan to a specific user
  • Select the user who will receive the subscription
  • Choose an active subscription plan
  • Set the status (usually 'active' for immediate access)
  • Start date defaults to now if not specified
  • End date is auto-calculated for non-unlimited plans
Managing Subscription Plans
  • To create or edit subscription plans, use the section
  • Plans define pricing, duration, and features
  • This page only assigns existing plans to users
Status Options
  • Active: User has full access to premium challenges
  • Pending: Waiting for payment confirmation
  • Expired: Subscription has ended
  • Cancelled: Subscription was cancelled
Payment Information
  • Amount paid should match the plan price
  • Payment reference is optional but helpful for tracking
  • Use this for manual subscriptions or payment adjustments
